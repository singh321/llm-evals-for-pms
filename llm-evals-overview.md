# LLM Evaluation Overview

## What is LLM Evaluation?

Evaluating how well an AI system performs on:
- accuracy
- relevance
- usefulness
- safety

---

## Types of evaluation

### 1. Offline Evals
- done before release
- uses golden datasets

### 2. Online Evals
- done in production
- based on real user behavior

---

## Why evals are hard

- outputs are probabilistic
- multiple correct answers exist
- quality is subjective

---

## PM’s role

- define evaluation criteria
- align on quality thresholds
- connect evals to business metrics
