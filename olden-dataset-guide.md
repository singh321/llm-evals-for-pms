# Golden Dataset Guide

## What is a Golden Dataset?

A curated dataset used to evaluate model quality.

---

## Structure

Each example should include:
- input
- expected output
- evaluation criteria

---

## Example

Input:
"Summarize this meeting"

Expected Output:
- key decisions
- action items

---

## Best Practices

- cover edge cases
- include failure scenarios
- keep dataset evolving

---

## PM Responsibility

- define what “good” looks like
- ensure dataset reflects real use cases
